Student Management System
# Project Overview

The Student Management System is a web-based application built using JSP, Servlets, JDBC, and MySQL.
This system helps manage student records by allowing users to add, edit, delete, and view student information.
It is designed for learning and demonstration of Java Web Development concepts.

# Objectives

Add new student details

Update existing student information

Delete a student record

View all student records

Establish JDBC connectivity with MySQL

Use JSP for frontend and Servlets for backend processing

Maintain proper MVC architecture

# Features

Add Student

Update Student

Delete Student

View Students

MySQL database integration

Clean UI using HTML and CSS

# Technologies Used

Java (JSP + Servlets)

MySQL Database

JDBC Connectivity

Apache Tomcat Server

HTML & CSS for user interface
